Miro

Miro allows users to create interactive online whiteboard sessions. The main features of Miro are listed below:

  • Click and create toolbar menus - add sticky notes, cards, shapes, text boxes and free form drawings.
  • Embed other website contents using iframes - eg YouTube videos.
  • Create wireframe diagrams and flexible grids.
  • Use templates to create Kanban boards, mind maps, data charts and other process flow diagrams.
  • Configure boards user access: set read/edit/write access to boards, invite collaborators, lock board elements.
  • Create custom widgets using the Miro SDK.

Zoom Top Security Tips

Zoom is an easy way to set up safe and secure video-conferencing. Follow the guidelines below to keep yourself and clients protected.

  • Keep your installed version of Zoom up to date
  • Password protect your meetings and never share your meeting ID on public platforms.
  • Enable 'Require a password' for scheduled and instant meetings.
  • Consider letting Zoom create a random ID for each meeting instead of using your meeting ID.
  • Use a waiting room. This will allow you to screen participants before allowing them to join.
  • Restrict screen sharing to the host only (unless needed by others) and disable file sharing.
  • Configure your settings so that removed particants are unable to re-join a meeting (if you are concerned about continuing disruption).
  • Be cautious about clicking on what look like Zoom links in emails, they may be phishing attacks.

Since April 4th 2020, Zoom has enabled the waiting room feature and additional passcode settings for all Basic users on free accounts and accounts with a single licensed user.

Setting zoom up for local recordings with screen sharing and all participants - Quick Guide

Please note, screen sharing has an influence on how the Zoom interface is presented. This guidance is based on the interface without screen sharing.

Set up the online Zoom portal

  1. Sign into Zoom at: https://zoom.us/
  2. Click on the Settings option on the left hand menu bar.
  3. Click on the Recording tab in the main page section.
  4. If Cloud Recording is selected, disable it by clicking on the right hand toggle button.
  5. Enable Local Recording by clicking on the right hand toggle button.
  6. Sign out of the online Zoom portal.

Adjust local Zoom settings

  1. Bring up the Zoom application on your PC/tablet.
  2. Sign in with the same credentials that you use for the online portal.
  3. Click the settings icon at the top right of the window - the settings pop-up should open.
  4. Click the Recording option in the left hand menu.
  5. Go into the Recording section and choose your options.
    • If preferred, choose a different location for saving files (click the Change button). A default location is shown e.g. 'H:\Zoom'.
    • You can choose to record separate audio files for each participant.
    • You can choose to add a time stamp to the video.
    • Make sure to select 'Record video during screen sharing'.
  6. When you have finished selecting your Recording options, close the settings pop-up by using the X in the top right corner.
  7. You can now close the Zoom application pop-up in the same way.
